# Football AI V7

ແອັບ Streamlit ສຳລັບດຶງຂໍ້ມູນບານ ແລະ ວິເຄາະ 1X2 / AH / OU

## ວິທີໃຊ້

1. ສ້າງ repository ໃໝ່ໃນ GitHub
2. ອັບໂຫຼດໄຟລ໌ `app.py`
3. ສ້າງໄຟລ໌ `requirements.txt`
4. ນຳ repo ໄປ deploy ໃນ Streamlit
5. Main file path ໃຫ້ໃສ່ `app.py`

## ໄລບຣາຣີທີ່ໃຊ້

- streamlit
- pandas
- requests
- lxml
- html5lib

## ໝາຍເຫດ

- ຖ້າ app error ໃຫ້ກວດເບິ່ງວ່າ code ໃນ `app.py` ຖືກວາງຄົບທຸກບັນທັດ
- ບໍ່ໃຫ້ວາງ code ແບບຕິດກັນໃນບັນທັດດຽວ
- ຫຼັງຈາກ commit ແລ້ວ ໃຫ້ Reboot app / Redeploy
